Ways to Center a Div

1. Using Flexbox

    Container:
  • display: flex;
  • justify-content: center; // horizontally center
  • align-items: center; // vertically center
content

2. Using Grid

    Container:
  • display: grid;
  • place-items: center;
content

3. Using Absolute Positioning

I don't recommend this method, because it's not responsive. It works well only for fixed height and width containers.
    Container:
  • position: absolute;
    • Content:
  • position: absolute;
  • top: 50%;
  • left: 50%;
  • transform: translate(-50%, -50%);
content

4. Using Table Display

I also don't recommend this method, you can't control the height of the content box.
    Container:
  • display: table;
    • Content:
  • display: table-cell;
  • vertical-align: middle;
  • text-align: center;
content
