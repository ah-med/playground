Using Cookies to Manage User Sessions

How it works

  1. On login page.
  2. User fills out the login form and clicks on the login button.
  3. The server will set a sessionId cookie in the browser.
  4. The browser redirects the user to the dashboard page.
  5. On dashboard page reload, the browser will check if the user has an active session.
  6. If no active session, browser redirects to login page.
